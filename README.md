# Personal Habits Tracker

A simple and clean web app to help you build better habits and stay consistent, developed from scratch using React. Easily track your daily habits, mark completion, and stay focused on self-improvement.

---

# Features

-  Add new habits with ease
-  Track daily habit completion
-  View historical progress
-  Edit and add habits
- See quick stats and daily summaries
-  Fully responsive for mobile and desktop
